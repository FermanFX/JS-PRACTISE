Playwrite PL Variable Font
==========================

This download contains Playwrite PL as both a variable font and static fonts.

Playwrite PL is a variable font with this axis:
  wght

This means all the styles are contained in a single file:
  PlaywritePL-VariableFont_wght.ttf

If your app fully supports variable fonts, you can now pick intermediate styles

  static/PlaywritePL-Light.ttf
  static/PlaywritePL-Regular.ttf
